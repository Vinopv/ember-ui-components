# Ember-ui-components

This ember-cli addon is a collection of common user interface components.

## Demo

http://lozjackson.github.io/ember-ui-components/

## Installation

* `npm install ember-ui-components --save-dev`

## Components

* CheckBox
* Select
* Button
* SlideMenu
* ContextMenu
* Dropdown
* ModalWindow
* ModalDialog

## Services

* Dialog

## CSS Classes

* Button
* Panel
