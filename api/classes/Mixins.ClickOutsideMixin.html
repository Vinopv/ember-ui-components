Module: ember-ui-components

Click Outside Mixin

This mixin attaches event listeners when the consuming component is inserted into the DOM and provides a method for handling clicks outside the target element.

import Ember from 'ember';
import ClickOutsideMixin from 'ember-ui-components/mixins/click-outside';

export default Ember.Component.extend(ClickOutsideMixin, {

  handleClickOutside(event) {
    // handle the click event...
  }

});

targetElement

Defined in addon/mixins/click-outside.js:37

Target Element

This element is the target element which is used to determine if the click is inside or outside.

By default this is an alias of element, which means that the targetElement is this component (the component that has consumed the ClickOutsideMixin).

You may want to set the targetElement to something else. For example, the parent element:

targetElement: Ember.computed.alias('element.parentElement')
