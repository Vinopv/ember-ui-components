setPosition

Set the left/top css properties of an element.

element should be a reference to an HTML element. Either a string selector that can be used with jQuery, or a jQuery selection object.

If position is not specified, then the current mouse position will be used.

position should be an Ember.Object with x and y properties. Both x and y should be numbers

Parameters:

  • element String | Object
  • position Object
